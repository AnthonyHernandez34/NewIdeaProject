HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

Metadata Metadata, or extended information, can be provided via the element within the document's A few examples of things specified via metadata are: search keywords page description page author page character set.

HTML element is a component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document. HTML elements can contain formatting instructions, semantic meaning, and content.
HTML form is a section of a document containing normal content, markup, special elements called controls (checkboxes, radio buttons, menus, etc.), and labels on those controls.

CSS stands for Cascading Style Sheets. CSS allows us to customize the presentation of a web page by specifying the colors, fonts, layout, and more. CSS also provides for the separation of the document content from the presentation. This separation has many benefits, including:
  1. Content can be used independently of styling for other purposes.

  2. Styles can be re-used across pages.

  3. Updating styles in a stylesheet that is used in many pages is much more efficient than updating all the pages individually.

  4. Bandwidth savings can be significant due to stylesheet caching.

Inline/Embedded Stylesheet Another method of adding styles to a webpage is via an embedded stylesheet. This can be accomplished by using a style element in the head of your HTML page.
External Stylesheet Finally, another way to add styles to a web page is to link to an external stylesheet.

JavaScript, abbreviated JS, is a high-level, interpreted computer programming language that can be executed within a web browser
function is a reusable block of code that performs a specified task, usually taking an input and producing an output. We have used several different functions already, and in this lesson we will learn how to define and use our own functions
The if/else statement is a part of JavaScript's "Conditional" Statements, which are used to perform different actions based on different conditions.
  1. Use if to specify a block of code to be executed, if a specified condition is true

  2. Use else to specify a block of code to be executed, if the same condition is false

  3. Use else if to specify a new condition to test, if the first condition is false

  4. Use switch to select one of many blocks of code to be executed

Unit tests are functions that test the expected vs. actual output of functions when provided various inputs. Writing unit tests is a way of authoring automated tests where you have code that checks to make sure that your implementation code works as intended.

Loops are used to execute the same block of code again and again, as long as a certain condition is met. The basic idea behind a loop is to automate the repetitive tasks within a program to save the time and effort. JavaScript now supports five different types of loops:

  1. while — loops through a block of code as long as the condition specified evaluates to true.

  2. do…while — loops through a block of code once; then the condition is evaluated. If the condition is true, the statement is repeated as long as the specified condition is true.

  3. for — loops through a block of code until the counter reaches a specified number.

  4. for…in — loops through the properties of an object.

  5. for…of — loops over iterable objects such as arrays, strings, etc.
